# FairSSL Website

Source code for the FairSSL project website.

Website: https://icml26-fairssl.github.io/

Paper: `static/pdfs/FairSSL_paper.pdf`  
Poster: `static/poster/FairSSL_poster.pdf`  
Code: https://github.com/abtinmU/FairSSL

This is a static GitHub Pages site. To publish it, place `index.html`, `README.md`, and the `static/` directory at the root of the `icml26-fairssl.github.io` repository and deploy from the `main` branch root.

Website design adapted from Nerfies.
